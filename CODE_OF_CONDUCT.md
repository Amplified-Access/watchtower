# Code of Conduct

AmplifiedAccess has adopted a Code of Conduct that we expect all community members, contributors, and project participants to adhere to.

Please read the full text at [thewatchtower.tech/code-of-conduct](https://thewatchtower.tech/code-of-conduct).

## Reporting

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to:

- [noble@amplifiedaccess.org](mailto:noble@amplifiedaccess.org)
- [aziz@amplifiedaccess.org](mailto:aziz@amplifiedaccess.org)

All reports will be reviewed and investigated promptly and fairly.
